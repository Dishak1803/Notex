# NoteX ??
A study companion web app for engineering and medical students.

## Features
- Upload and share notes, slides, previous papers
- Study planner
- Leaderboard and streaks
- Bookmarks and comments
- MCQ quiz game

## Setup
1. Clone the repo
2. Run npm install
3. Copy .env.example to .env and fill in your credentials
4. Run node server.js

## Tech Stack
- Node.js + Express
- MySQL
- Socket.io
- Groq AI
